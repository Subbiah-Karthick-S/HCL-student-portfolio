1. Non-Academic Credit Calculator
This project is designed to calculate and manage credit points for students based on extracurricular activities, certifications, and achievements. 
It provides a structured system to track and evaluate non-academic performance. 
This project was recognized as an innovative idea and submitted for patent consideration.

2. AI-Powered Study Planner
This project helps students plan their study schedule efficiently by considering subject difficulty, available time, and exam deadlines. 
It automatically generates an optimized timetable and tracks progress. 
It also supports exporting study plans as PDF for better usability.

3. BALANCIFY – AI-Powered Budgeting Assistant
BALANCIFY is a smart budgeting tool that analyzes user spending patterns using AI. 
It provides insights, visualizations, and allows users to simulate different financial scenarios using “What-If” analysis. 
The focus is on privacy and user-friendly financial planning.

4. Pharmasense – AI-Based Healthcare Platform
Pharmasense is a secure and user-friendly healthcare platform designed for doctor–patient interaction, appointment booking, and online consultations. 
It integrates AI-based prescription analysis using OCR and NLP to extract and simplify medical information. 
The system also uses machine learning (Random Forest algorithm) to predict treatment outcomes based on patient data. 
Additionally, it provides visual insights and maintains centralized digital health records to track patient progress and treatment history.
